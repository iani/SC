BMSpeaker represents a speaker connected to a hardware output bus


Inherits from: Object


BMSpeaker represents a speaker connected to one of a server's hardware outputs. In addition to the output bus index, it encapsulates information on location (in both Cartesian and spherical coordinates), speaker type, trim values, and directivity.


See also: BMInOutArray, BMSpeakerSpec, BMAutoBalancer, BMSpeakerArrayGUI, BMSpeakerListVisualiser


Compatibility and Polymorphism


BMSpeaker responds to asUGenInput, asControlInput, asOSCArgBundle, and asOSCArgArray by returning its bus index. It can thus be used directly in most places where you would pass such an index. For example:


i = BMSpeaker(\right, 1);

// make a Synth

x = {Out.ar(i, SinOsc.ar(440, 0, 0.2))}.play; // pass BMSpeaker as bus arg

x.free;


BMSpeaker masquerades as an Association, returning its name as key, and itself as value. The standard Collection class to use for speaker lists is BMInOutArray.


Creation / Class Methods


*new (name, index, x, y, z, spec)

Create a new BMSpeaker using Cartesian coordinates. (0,0,0) is generally considered to be head height at the centre of the listening area.

name - A Symbol corresponding to the name of this speaker.

index - The bus index corresponding to this speaker's hardware output.

x - x (left-right) coordinate in meters. Default value is 1. 

y - y (front-back) coordinate in meters. Default value is 1.

z - z (height) coordinate in meters. Default value is 1.

spec - A Symbol corresponding to the name of an existing BMSpeakerSpec. Default value is 'Generic'.

b = BMSpeaker.new(\left, 0, -1, 1.732, 0, 'UREI-809'); // make a "new" Urei 809

b.azi; // spherical coords automatically calculated



*newFromSpherical (name, index, azi, ele, rad, spec)

Create a new BMSpeaker using spherical coordinates, generally considered relative to (0,0,0) in Cartesian coordinates (see above).

name - A Symbol corresponding to the name of this speaker.

index - The bus index corresponding to this speaker's hardware output.

azi - Azimuth (horizon) angle in degrees. 0 is straight ahead (positive y-axis). Default value is 0.

ele - Elevation (height) angle in degrees above or below the azimuth plane. Default value is 0.

rad - Radius in meters from the centre point.

spec - A Symbol corresponding to the name of an existing BMSpeakerSpec. Default value is 'Generic'.

b = BMSpeaker.newFromSpherical(\left, 0, -30, 0, 2, 'Kef-C20'); // make a "new" Kef C20

b.x; // Cartesian coords automatically calculated



Accessing Instance and Class Variables

index_

index

Set or get this speaker's bus index.

description_

description

Set or get a String to serve as a human readable description of this speaker. This can be more verbose than name. Default value is nil.

name_

name

Set or get a Symbol to use as this speaker's name. In a BMInOutArray this will serve as this speaker's key.

autoTrim_

autoTrim

Set or get value in dBFS to use as trim setting for this speaker. BMTrimPluginsRack has an autoTrim method which queries this value. Generally this is populated using BMAutoBalancer. Default value is nil.

spec_

spec

Set or get an instance of BMSpeakerSpec which describes the characteristics of this speaker's model.

directivity_

directivity

Set or get a Symbol used to describe this speaker's directivity, i.e. is it pointed towards the listener or reflected. Valid settings are 'direct' and 'reflected'. Default value is 'direct'.


Setting coordinates


Setting any Cartesian coordinate will result in the spherical coordinates being updated, and vice versa. 


Naturally users are free to use coordinates according to any reference desired. Cartesian coordinates and rad can be in any units desired, but azi and ele must be in degrees since this is assumed in the update calculations. In general however other elements of the library assume that Cartesian coordinates are in meters and that both Cartesian and spherical coordinates are relative to the Cartesian point (0,0,0), which indicates head height at the centre of the listening area.


x_

x

Set or get this speaker's x coordinate (usually in meters).

y_

y

Set or get this speaker's y coordinate (usually in meters).

z_

z

Set or get this speaker's z coordinate (usually in meters).

azi_

azi

Set or get this speaker's azimuth angle in degrees.

ele_

ele

Set or get this speaker's azimuth angle in degrees.

rad_

rad

Set or get this speaker's radius  (usually in meters).

Other Instance Methods


asUGenInput

Returns this speaker's bus index.



asControlInput

Returns this speaker's bus index.



isBMSpeaker

Returns true. An extension to class Object defines this message to return false.



key

This is a synonym for name, allowing a BMSpeaker to masquerade as an Association.



isValidBMInOutArrayMember

Returns true. An extension to class Association defines this message to return true. An extension to class Object defines this message to return false.



Examples


// 5.1 Array (-0.1)

(

l = [[-30, 'L'], [30, 'R'], [0, 'C'], [-110, 'Ls'], [110, 'Rs']].collectAs({|el, i|

BMSpeaker.newFromSpherical(el[1], i, el[0], 0, 2);

}, BMInOutArray);

)

BMSpeakerListVisualiser(l); // let's look at it


// 8 Channel Ring

(

l = [[0, 'C' ], [45, 'FR'], [90, 'R'], [135, 'BR'], [180, 'B'], [-135, 'BL'], [-90, 'L'], [-45, 'FL']].collectAs({|el, i|

BMSpeaker.newFromSpherical(el[1], i, el[0], 0, 2);

}, BMInOutArray);

)

BMSpeakerListVisualiser(l); // let's look at it


// zig zag partial dome

(

l = [[-22.5, 40, 'Upper1'], [22.5, 40, 'Upper2'], [-67.5, 40, 'Upper3'], [67.5, 40, 'Upper4'], [-112.5, 40, 'Upper5'], [112.5, 40, 'Upper6'], [-157.5, 40, 'Upper7'], [157.5, 40, 'Upper8'], [-45, 0, 'Lower1'], [45, 0, 'Lower2'], [-90, 0, 'Lower3'], [90, 0, 'Lower4'], [-135, 0, 'Lower5'], [135, 0, 'Lower6'], [0, 0, 'Lower7'], [180, 0, 'Lower8']].collectAs({|el, i|

BMSpeaker.newFromSpherical(el[2], i, el[0], el[1], 2);

}, BMInOutArray);

)

BMSpeakerListVisualiser(l); // let's look at it
