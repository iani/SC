BMBoid represents a single boid in a BMBoidSpace


Inherits from: Object


BMBoid represents a single boid (i.e. flock member) in a BMBoidSpace. Normally you would not create these directly, rather they are created and controlled by BMBoidSpace for you. As such all its methods should be considered 'private'. See BMBoidSpace for examples.


N.B. This implementation is still experimental and should be considered subject to change in future versions of the library.


See also: BMBoidSpace, BMSwarmGranulator


Instance Methods


pos

Returns this boid's current position as an Array of the same size as this boid's BMBoidSpace's number of dimensions, e.g. [x, y, z] for a three dimensional space.
